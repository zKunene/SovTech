The application queries data from the "https://swapi.dev/" and outputs that data as a list of character names.

The applications allows users to search for a string in the list, which will return a dropdown of matches.
Also, users can search for a page and the characters on that page will apear.

To view more details on a name in either search bar, click on the name to display more information; double click the name to hide the information.

To remove results from page search, search invaled number(i.e 0)

Click Search once to display names matching query. Double click search to empty the text field and double click clear to to no longer display the results of the search.
